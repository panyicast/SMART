# 卫星变轨策略研究文档

## 1. 引言
本研究针对一颗设计重量为 6500kg 的卫星，其定点位置为 120°E，在轨服务寿命为 15 年。原计划于 2026 年 3 月使用 LM5B 火箭在文昌发射中心发射，现推迟至 2026 年 5-6 月间发射。研究目标是通过变轨策略，将卫星从初始超同步转移轨道调整到目标地球同步轨道，并尽量降低推进剂消耗。

## 3. 变轨策略

### 3.1 策略说明
卫星通过 4 次远地点变轨加 1 次近地点点火，实现倾角从初始 16.5° 变为目标地球同步轨道倾角 6°。每次点火包括发动机与小型推力器操作。

### 3.2 发动机与推力器参数

|------|------|
| 发动机推力 | 490 N |
| 发动机比冲 | 314.1 s |
| 推力器推力 | 10 N |
| 推力器比冲 | 290 s |

## 4. 卫星运动方程

在赤道坐标系内，卫星运动方程为：

```
(d/dt) [r, v]^T = ...
```

其中，\(r\) 表示位置向量，\(v\) 表示速度向量，f 为发动机推力向量。

## 6. 推进剂消耗
总点火时间约 288.4 min，推进剂消耗（包括推力器和发动机点火）约 2614.8 kg。

## 7. 总结
通过五次变轨，卫星最终进入准同步轨道，实现初始倾角 16.5° 调整至 6°，同时有效控制了推进剂消耗。
